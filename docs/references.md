# References

## Primary Sources

**Schlesinger, Don.** *Blackjack Attack: Playing the Pros' Way.* 3rd ed. RGE Publishing, 2005.
BJW references this book for DI, SCORE, N₀, deviation, and blackjack simulation literature. BJW's current variable-betting variance decomposition is a BJW implementation choice; it is not currently presented as a direct quoted equation from this book.

**Griffin, Peter A.** *The Theory of Blackjack: The Compleat Card Counter's Guide to the Casino Game of 21.* 6th ed. Huntington Press, 1999.
Foundational mathematical treatment of blackjack probability, house edge, and card counting theory.

**Wong, Stanford.** *Professional Blackjack.* Pi Yee Press, 1994.
Reference for basic strategy tables, counting system applications, and practical advantage play methodology.

## External Tools Used as References

**Wattenberger, Norm.** *CVData / CVCX.* Casino Verite Software.
Used as the primary external comparison reference throughout BJW's validation campaign. CVData is a closed-source commercial simulator. BJW's output was compared against CVData across 8 core scenarios at 1 billion rounds each. Results are published in the [Validation Position](validation/position.md) page.

Where BJW and CVData disagree, see the validation position page for the full delta table, root cause classification, and our interpretation.
